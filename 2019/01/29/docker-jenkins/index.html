docker jenkins

1拉取镜像
2根据镜像启动容器
3根据提示去查需要初始的密码
4根据界面提示去安装插件
5安装ssh key连接gitLab

这个时候要注意，

  1. 配置的公钥私钥来自于jenkins服务器,注意用jenkins
    账户生成公钥私钥。
  2. 在gitLab的deploy key里配置公钥
  3. 在jenkins凭据里配置私钥
  4. 然后在jenkins服务器上git clone 下地址，用来测试是不是可以连接到gitLab.
    如果可以证明配置成功。
